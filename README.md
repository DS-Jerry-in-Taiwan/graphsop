# graphsop

TwinLex (GraphSOP) - 智慧攝影機數位孿生知識建模與閉環控制專案

## 專案簡介
本專案以 Python、Streamlit、LlamaIndex 等技術，實現智慧攝影機的屬性圖知識建模、SOP 流程規劃、閉環驗證與可視化前端。

## 主要功能
- 屬性圖知識建模與自動化提取
- SOP 流程規劃與驗證
- Mock Camera 虛擬硬體層
- Streamlit 前端互動與即時狀態回饋
- E2E 測試與驗收流程
