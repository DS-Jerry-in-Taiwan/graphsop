# TwinLex Streamlit 操作手冊

## 啟動前端
1. 啟動虛擬環境：
   ```
   source .venv/bin/activate
   ```
2. 啟動 Streamlit 前端：
   ```
   streamlit run src/ui/app.py
   ```
3. 瀏覽器將自動開啟，或手動訪問 http://localhost:8501

## 操作流程
### 1. 指令輸入
- 在主畫面輸入框輸入指令（如：「旋轉到 30 度」、「變焦 5 倍」），點擊「送出」。

### 2. 狀態查詢
- 左側邊欄會即時顯示 Mock Camera 的 pan/tilt/zoom/power 狀態。

### 3. 驗證流程
- 每次操作後，主畫面會顯示 SOP 步驟、執行結果與驗證回饋。
- 若指令超過物理極限（如「旋轉到 400 度」），會顯示「驗證失敗」或「拒絕執行」。
- 若遇到隨機硬體故障（5% 機率），會顯示「驗證失敗」訊息。

### 4. 手動刷新
- 點擊側邊欄「手動刷新狀態」按鈕，可即時更新設備狀態。

## 驗證建議
- 測試合法指令（如「旋轉到 30 度」），確認狀態同步與驗證成功。
- 測試違規指令（如「旋轉到 400 度」），確認 UI 能正確攔截並顯示錯誤。
- 多次操作，觀察隨機故障與異常回饋。

## 常見問題
- 若無法啟動 Streamlit，請確認已安裝 streamlit 套件並啟用虛擬環境。
- 若 UI 無法即時更新，請手動刷新或重啟 App。
